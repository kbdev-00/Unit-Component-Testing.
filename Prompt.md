# AI Prompts Used – Prodesk IT Internship (Week 11)

The following AI prompts were used during the development of this project for learning and guidance purposes only.
All generated code was reviewed, understood, and manually integrated.

---

### Prompt 0 (Project Planning & Setup)

Create a step-by-step roadmap to implement unit and component testing in a React/Next.js application using Jest and React Testing Library.

---

### Prompt 1 (Testing Setup)

How do I install and configure Jest and React Testing Library in a React or Next.js project?

---

### Prompt 2 (Basic Component Testing)

Write a unit test for a simple React Button component to check if it renders correctly and displays the correct text passed via props.

---

### Prompt 3 (Multiple Component Testing)

How can I write unit tests for multiple UI components like Card, Input, and Button using React Testing Library?

---

### Prompt 4 (Interaction Testing)

How to test user interactions like button clicks and input field changes using userEvent or fireEvent in React Testing Library?

---

### Prompt 5 (Form Testing)

Create test cases for a form component to check if input values update correctly and submit actions work as expected.

---

### Prompt 6 (API Mocking)

How to mock API calls (fetch or axios) in Jest so that tests do not make real network requests?

---

### Prompt 7 (Async Testing)

How to test components that fetch and display data asynchronously using waitFor or findBy methods?

---

### Prompt 8 (Test Coverage)

How can I generate a test coverage report using Jest and ensure at least 70% coverage?

---

### Prompt 9 (Best Practices)

What are the best practices for writing clean, maintainable, and scalable test cases in React applications?

---

### Prompt 10 (Debugging Tests)

How to debug failing test cases in Jest and identify issues in React components?
